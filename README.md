AdNabu QA Engineer Assignment
Project Overview

This project automates a user scenario on the AdNabu Test Store using Python and Selenium.

Scenario Covered
Search for a product
Select the first product from results
Add the product to cart
Verify that the product is added successfully
Tech Stack
Python 3.x
Selenium WebDriver
Setup Instructions
Clone the repository:
git clone https://github.com/nunnavishal/vishal-assignment.git
Navigate to project folder:
cd AdNabu_Assignment
Install dependencies:
pip install -r requirements.txt
Run the script
